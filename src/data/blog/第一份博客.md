# 我的第一篇博客

你好，这里是段清衡。

折腾了几天 Astro、i18n、UI 细节之后，这个博客终于能跑起来了

这篇文章，就当作一个起点。

---

## 为什么要写博客？

我开始写博客，不是为了展示，而是为了记录。

在学习技术的过程中，我发现：

- 很多知识学了就忘
- 很多问题解决后没有沉淀
- 很多想法没有地方表达

所以我决定开始写博客。

---

## 这个博客会写什么？

主要会记录：

- 前端学习过程（Astro / React / CSS）
- 项目实践
- 遇到的问题与解决方案
- 一些思考和总结
- agent学习

---

## 关于现在的我

我是一个前端 freshman，正在尝试建立自己的技术体系。

这个博客，是我成长的记录。
